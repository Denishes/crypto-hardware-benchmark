# AES Hardware Benchmark

This section contains FPGA and ASIC implementations of AES.

## Supported Key Sizes

- AES-128
- AES-192
- AES-256
- AES-512
- AES-1024

## Supported Architectures

- Pipeline
- Pipeline GF
- Iterative
- Iterative GF
- Partial Unrolled
- Partial Unrolled GF

## Targets

### ASIC
- Nangate45
- Future technologies

### FPGA
- ECP5
- ZCU102
- Future devices
